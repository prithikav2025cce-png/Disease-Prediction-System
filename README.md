# AI Symptom Checker

A Flask-based web application that predicts potential diseases based on user-selected symptoms using a weighted AI model.

## How to Run

1.  **Install Dependencies**:
    Make sure you have Python installed, then run:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Start the Server**:
    ```bash
    python app.py
    ```

3.  **Open the App**:
    Navigate to `http://127.0.0.1:5000` in your web browser.

## Features
- Searchable symptom list.
- Real-time selection counter.
- AI-driven disease matching (Mock Model).
- Interactive results modal with confidence scores.
